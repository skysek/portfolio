Développer une application web avec Java EE

Une application web désigne un logiciel applicatif hébergé sur un serveur et accessible via un navigateur web.

La création d’une application web peut se faire à l’aide du langage de programmation Java avec l’utilisation du Java EE (Java Enterprise Edition).

Java EE une extension de la plate-forme standard Java SE (Java Standard Edition) basée sur du Java. L’extension a pour but de faciliter le développement d’applications web en y ajoutant un nombre importants de fonctionnalités non disponibles en version standard.
